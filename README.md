### Prerequisites

Make sure you have **GHC (Glasgow Haskell Compiler)** or **GHCup** installed:

```bash

```

You can install via [GHCup](https://www.haskell.org/ghcup/):

```bash

```

### Running Code

To compile and run a file:

```bash

./output
```

Or run directly with `runghc`:

```bash
runghc filename.hs
```
